# Demo Script

1. Open the Advisor tab, select **Mitumba stall**, location **Gikomba**, describe the stall, and add current goals.
2. Submit to view the four advice cards (pricing, branding, competition, marketing) and point out actionable tips.
3. Switch to Business Plan, enter a KES 10,000 budget with current skills, and generate the plan.
4. Go to Cashflow, add two expenses (e.g., rent, stock) and one income; switch summary range to weekly.
5. Toggle browser offline, add another entry (it queues), then come back online to show it syncing automatically.
